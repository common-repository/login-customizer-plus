== Description ==

Login Customizer Plus plugin allows you to easily customize your wordpress login page.
You can customize almost anything and make it look the way you want. 

Easy and Useful.

Login Customizer Plus allows you to change almost any aspect of the login page:
= General =
* Background Image Customization
* Background Color Customization
* Custom Icon
* Add Social Media Icons
* Change Social Media Icons Color
* Many More are coming in future release...

== Installation ==

1. Install 'login-customizer-plus' via the plugin directory or Upload the entire 'login-customizer-plus' folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. You will find 'Login Customizer Plus' menu in your WordPress admin panel.
4. Then set your required plugin setting 

For basic usage, you can also have a look at the [plugin web site]( https://techuptodate.com.au/login-customizer-plus/ ).
